### ABot-AgentOS 发布：通用机器人 Agent OS 携终身多模态记忆，LoCoMo 达 87.5

ABot-AgentOS 是一个运行在底层控制器之上的通用机器人 Agent 操作系统，提供场景条件规划、上下文隔离技能执行、多阶段验证和多模态记忆等运行时层。它引入 Universal Multi-modal Graph Memory，将对话、视觉观察、空间关系等转为类型化节点与边，实现持久化源锚记忆。同时推出 EmbodiedWorldBench 基准，包含 16 个室内外场景、四个难度级别和超过 200 个任务。

### ABot-N1：慢-快架构解耦认知与控制，城市导航POI到达率跃升35%至77.3%

ABot-N1提出一种慢-快架构，将视觉语言导航的推理与动作分离。慢速推理模块通过思维链生成像素级目标锚点，作为跨任务通用接口；快速动作专家利用文本与像素双引导，以原生控制频率输出连续航点。该模型在仿真与真实环境中刷新多项记录：城市级POI到达率提升35.0%达77.3%，复杂室内外场景成功率分别达95.4%和92.9%，并在物体到达、人员跟随、指令跟随等任务中保持鲁棒。

### LightMem-Ego：轻量级多模态记忆系统，让AI记住你的日常生活

LightMem-Ego是一个轻量级流式多模态记忆系统，专为手机和可穿戴设备设计。它持续捕捉用户的第一视角视觉和音频流，将其对齐到统一时间线，并组织成当前、短期和长期分层记忆。用户提问时，系统动态路由到对应记忆层级，生成基于多模态证据的答案。演示已可部署在智能手机和AI眼镜上，支持找物、对话回忆、生活总结、日常发现和个性化辅助。对开发者而言，这意味着低成本的个人长期记忆方案，有望推动真正理解用户生活的AI助手落地。

#多模态记忆 #AI助手 #可穿戴设备

### Cline CLI v3.0.40：新增 API Key 手动输入与登录稳定性修复

Cline 发布 CLI v3.0.40 版本，核心更新包括为 OAuth 提供商增加手动输入 API Key 的备用通道，修复了切换模型时配置不重载及网络波动导致的意外登出问题。同时优化了 Bun 全局安装检测、会话 ID 保持及上下文压缩预算处理。对开发者而言，手动密钥输入提升了本地部署灵活性，而登录与会话稳定性的改进能显著减少编码中断，保障长任务连续性。
