### 阿里开源Page-Agent：用自然语言控制网页的JavaScript GUI代理

阿里巴巴在GitHub上开源了Page-Agent，一个基于JavaScript的页面内GUI代理。它允许开发者通过自然语言指令操控网页界面，无需手动编写交互脚本。该工具直接在浏览器中运行，降低了自动化任务的实现门槛。对于前端、测试和流程自动化从业者，这意味着可以更便捷地构建网页操作代理，例如用自然语言执行表单填写、数据抓取或回归测试，减少对DOM细节的依赖。

### Garry Tan 公开 Claude Code 23 工具配置，一人包揽 CEO、设计、QA 全栈角色

Garry Tan 在 GitHub 发布了 gstack 仓库，分享了他为 Claude Code 定制的 23 个工具，这些工具分别承担 CEO、设计师、工程经理、发布经理、文档工程师和 QA 等职责，形成一套完整的多角色开发流程。从业者可直接复用这套配置，将 Claude Code 从单一助手升级为覆盖产品全周期的协作团队，加速从想法到交付的整个过程。
