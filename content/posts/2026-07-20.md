### RAGU开源：7B小模型+多步GraphRAG，性能超Qwen2.5-32B

Hugging Face论文2607.11683发布开源引擎RAGU，采用两阶段提取与DBSCAN去重等模块化流程解决GraphRAG噪声问题。其核心亮点是训练了7B参数模型Meno-Lite-0.1，专注语言技能而非世界知识，在知识图谱构建上显著优于Qwen2.5-32B，并在医疗基准测试中超越HippoRAG2。该方案支持pip安装且单卡即可运行，为开发者提供了高效、低成本的GraphRAG落地新路径。

### RESOURCE2SKILL：从视频、代码等多模态资源蒸馏可执行技能，提升11.9%

### 小米发布Xiaomi-Robotics-1：10万小时真实轨迹训练VLA模型

小米发布Xiaomi-Robotics-1，一个视觉-语言-动作（VLA）基础模型。该模型在超过10万小时真实世界操作轨迹上预训练，具备零样本执行多种移动操作任务的能力，并可通过少量数据微调适应新任务。其两阶段训练方案（预训练+后训练）展现出强缩放行为，数据量和模型规模越大性能越好。对从业者而言，这意味着机器人基础模型走向规模化，可降低下游任务数据收集成本，推动机器人泛化能力提升。

### Apple 提出 LenVM：在 Token 级别建模生成长度，更精细控制推理成本

苹果机器学习团队推出 Length Value Model (LenVM)，将生成长度建模转化为 token 级别的价值估计问题。每个解码步骤预测剩余长度，并对每个 token 分配恒定负奖励。这使从业者能更细粒度地调控推理成本与性能，例如提前终止生成或动态调整策略，为高效推理开辟新路径。

#Apple #长度建模 #Token级别
依据：[LenVM 是一个 token 级别的框架，在每个解码步骤建模剩余生成长度。](https://machinelearning.apple.com/research/length-value-model)；[该方法将长度建模表述为价值估计问题，并对每个生成的 token 分配恒定负奖励。](https://machinelearning.apple.com/research/length-value-model)
来源 [apple-ml](https://machinelearning.apple.com/research/length-value-model) · 93 分
