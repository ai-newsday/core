### Embodied.cpp 发布：为异构机器人上的具身AI模型打造便携推理运行时

具身AI模型（如VLA和WAM）部署时面临碎片化问题。新发布的Embodied.cpp是一个便携式C++推理运行时，通过五层架构（输入适配器、序列构建器、骨干执行、头插件、部署适配器）实现模块化多速率执行和延迟优先融合推理。在HY-VLA和pi0.5上任务成功率分别达100%和91%，WAM基准测试中块内存从312.2 MiB降至88.1 MiB。从业者可直接将其用于异构机器人部署，无需为每款模型编写胶水代码，显著提升效率。

### LangChain 发布 OpenRouter 0.2.6：支持自定义 HTTP 头，模型配置刷新

LangChain 发布 langchain-openrouter 0.2.6 版本，主要修复了 `default_headers` 参数，允许开发者在请求中注入自定义 HTTP 头信息，同时刷新了模型配置文件数据。这一更新让集成 OpenRouter 时能更方便地添加认证令牌或追踪标记，提升请求灵活性与可观测性，对于需要精细控制 API 通信的生成式 AI 应用场景尤为实用。

### Lobe Chat v2.2.10-canary.31：修复异构Agent配额缓存与文件代理访问

LobeHub 发布 v2.2.10-canary.31 版本，包含两项修复：一是修复异构 Agent (heterogeneous-agent) 的配额缓存问题，确保缓存按来源保留；二是撤销文件代理 (file-proxy) 对 `/f/:id` 的匿名公开访问，恢复更严格的权限控制。该版本为自动构建的金丝雀版本，仅供测试，不建议用于生产环境。开发者可借此提前验证缓存逻辑和文件访问策略的变化。
